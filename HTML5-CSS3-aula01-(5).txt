NO HTML:

<!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8"/>
	<title> Página de teste </title>
	<link rel="stylesheet" hreh="estilo.css"/>
</head>
<body>
	Texto <strong>qualquer</strong> para usar como exemplo deste curso maravilhoso.
</body>
</html>


<b></b> - negrito sem semântica

<strong></strong> - negrito em semântica de importância (deficiente)



- Colocar escrito em itálico -



<body>
	Texto <i>qualquer</i> para usar como exemplo deste curso maravilhoso.
</body>

<i></i> - itálico sem semântica

<em></em> - itálico em semântica de importância (deficiente)



- Tag para deixar escrito pequeno -



 <body>
	Texto qualquer para usar como exemplo deste curso maravilhoso
	<br/>
	<small>Segunda frase para exemplificar.</small>
</body>



- Tag para deixar escrito riscado -



<body>
	Texto qualquer para usar como <del>exemplo</del> deste curso maravilhoso
	<br/>
	Segunda frase para exemplificar.
</body>



- Tag para deixar escrito marcado como marca-texto -



 <body>
	Texto qualquer para usar como exemplo deste <mark>curso maravilhoso</mark>
	<br/>
	<Segunda frase para exemplificar
</body>

--- Formatação de textos no CSS (2/2) ---


NO CSS:


*{
	box-sizing:border-box;
}

body{
	margin:50px;
	font-size:17px;
}

p{
	color:#0000FF;
	font-size:20px;
	font-family:Helvetica, Arial, Verdana;
	text-align:left;
	(usar um de cada vez)text-transform:uppercase;
	letter-spacing:10px;
	word-spacing: 20px;
	line-height: 40px;
	text-shadow: 0px 2px 3px #FF0000;	
}

(uppercase - maiúsculo | lowercase - minúsculo)
(text shadow - primeiro item [0px] vai dizer o quanto a direita a sombra vai ficar | o segundo item [2px] é o quão pra baixo vai a sombra | o terceiro item [3px] é o quão esfumaçado vai ficr a sombra)
