--- Width/height no CSS ---

Width - Largura

Height - Altura 

- É possível definir altura e largura de um elemento em pixels e também em porcentagem.

- Inherit [É um valor que vai usar a largura ou altura de um item anteriors]

- max-width [vai limitar um tamanho máximo para um item]
- min-width [vai limitar um tamanho mínimo para um item]

	<meta charset="UTF-8"/>
	<title> Página de teste </title>
	<link rel="stylesheet" href="estilo.css"/>
</head>
<body>
	<h1>Página de Teste</h1>
	
	<a href="paginas/login.html">Fazer login</a>
</body>
</html>

No login.html:

<!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8"/>
	<title> Página de teste </title>
	<link rel="stylesheet" href="estilo.css"/>
</head>
<body>
	<h1>Página de Login</h1>
	
	<a href="teste.html">Ir para teste</a>
</body>
</html>

Dica:

<a href="teste.html" target="_blank">Ir para teste</a>

- O target="_blank vai abrir uma nova aba para acessar o link desejado

<a href="teste.html" title="Página do login">Ir para teste</a>

- vai dar um título quando passar com o mouse por cima do link (sem clicar)
